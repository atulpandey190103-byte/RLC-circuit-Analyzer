# RLC Circuit Analyzer

A Python-based tool to analyze RLC circuits and compute key parameters like impedance, phase angle, and resonant frequency.

## Features
- Computes impedance, reactance, and resonance
- Supports accurate electrical calculations
- CLI-based interface
- Unit tested using unittest

## How to Run

```bash
python main.py
