# ⚡ RLC Circuit Analyzer

A Python-based tool to analyze RLC circuits and compute key electrical parameters like impedance, reactance, phase angle, and resonant frequency.

---

## 🚀 Features
- Calculates Inductive Reactance (XL)
- Calculates Capacitive Reactance (XC)
- Computes Impedance (Z)
- Calculates Phase Angle
- Finds Resonant Frequency
- Input validation for accurate results

---

## 🛠 Tech Stack
- Python
- OOP (Object-Oriented Programming)
- CLI (Command Line Interface)
- Unit Testing (unittest)

---
